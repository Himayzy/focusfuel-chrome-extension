# 🚀 FocusFuel – Chrome Productivity Extension

FocusFuel is a lightweight Chrome extension designed to boost productivity by enabling Focus Mode and reducing digital distractions.

Built using **JavaScript** and **Chrome Extension API (Manifest V3)**.

---

## ✨ Features

- 🧠 Focus Mode toggle
- 🚫 Reduces distractions
- ⚡ Lightweight & fast
- 🎯 Simple and clean popup UI
- 🔒 Built with Manifest V3

---

## 🛠 Tech Stack

- JavaScript
- HTML
- CSS
- Chrome Extension API (Manifest V3)

---

## 📦 Installation Guide (Manual)

1. Download this repository as ZIP  
2. Extract the folder  
3. Open Chrome and go to:  chrome://extensions/
4. 4. Enable **Developer Mode** (top right)
5. Click **Load Unpacked**
6. Select the extracted extension folder

✅ Done! FocusFuel is now installed.
